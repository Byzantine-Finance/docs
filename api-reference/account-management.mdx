This section explains how to manage existing Byzantine accounts once KYC/KYB is complete. Account management covers configuration of an existing Byzantine account for day-to-day use.

| Capability           | Description                                                                                                                                                                                                                      |
| -------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Add bank account** | Associate a bank account with an account so that withdrawals can be sent to it (off-ramp). At least one bank account must be added before any off-ramp withdrawal can be initiated. Can have multiple bank accounts per account. |
