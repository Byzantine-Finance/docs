Byzantine has built an API to simplify the integration of our platform into your own.

<Info>
  Download the [OpenAPI specification](openapi-integrator.json) to view the full API endpoints.
</Info>

## API health

- `GET /v1/health` returns the service status.

## Customer management

- Retrieve details of a single user or entity.
- Create customers (user or entity) after obtaining the Bridge ToS acceptance link.

## Account management

- List bank accounts and account balances.
- Add bank accounts.
- Manage invitations: list, create, and send invitations to users.
- Manage user roles within accounts.

## Products

- Discover active products and their performance using the list, APY, and history endpoints.

## Transactions

- List and retrieve transaction details.
- **Passkey flow**: get unsigned payloads for account activation, deposits, and withdrawals, then sign and submit them.

## OTP authentication

- Initialize OTP for a user, authenticate with OTP code, and create passkeys via OTP session.
