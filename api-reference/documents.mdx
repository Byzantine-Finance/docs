title: "Accepted documents"
---

Depending on whether the user is an individual or a business entity, different documents are required for KYC and KYB.

We accept the following documents:

### ID documents

- Passport
- National ID card (back and front side)

### Proof of address

- Utility bills
- Recent bank statements
- Mobile phone bills
- Tax documentation

<Info>  
  All proof of address documents must be issued no more than 3 months ago.
</Info>
