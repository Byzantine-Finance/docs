### How to use the collection

1. Fork the collection
2. Insert your integrator private key in the `INTEGRATOR_PRIVATE_KEY` environment variable
3. Use the collection to test our API

<Note>
  By default, requests are sent to our sandbox API, ensuring no real data or live operations are impacted. To switch to the production API, selection the `Production` environment.
</Note>
