This section covers how to manage individual users after they have been created via the KYC/KYB flows.

You can:

- Inspect user details and verification status.
- Attach or rotate authenticators (passkeys) for a given user using the OTP-authenticated flow.
- Combine this with **Wallet authentication** to design your end-user security model.

Refer to the **User management** endpoints in the **API endpoints** section for exact request/response schemas.
