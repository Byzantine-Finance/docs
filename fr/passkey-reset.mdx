---
title: "Réinitialisation du passkey"
description: "Comment résoudre les problèmes de passkey et réinitialiser votre passkey avec l'aide de Byzantine."
---

Un passkey est une clé sécurisée stockée sur votre appareil qui vous permet d'approuver les transactions sur Byzantine, comme les dépôts et les retraits. Il fonctionne avec votre empreinte digitale, votre reconnaissance faciale ou le code PIN de votre appareil. Pour en savoir plus, voir [Comprendre les passkeys](/fr/documentation/account-management/passkeys).

Si vous perdez l'accès à votre passkey pour quelque raison que ce soit, cette page décrit les étapes de récupération. Si vous avez déjà essayé les étapes de dépannage ci-dessous et avez besoin d'une réinitialisation, contactez l'équipe Byzantine à [info@byzantine.fi](mailto:info@byzantine.fi) pour planifier un court appel.

## Ce qu'il faut essayer d'abord

La plupart des problèmes de passkey sont liés à l'utilisation du mauvais appareil ou du mauvais profil de navigateur. Avant de demander une réinitialisation, vérifiez les points suivants.

<AccordionGroup>
  <Accordion title="Actualiser la page et réessayer">
    L'invite du passkey peut parfois ne pas apparaître en raison d'un problème temporaire du navigateur. Fermez l'onglet, rouvrez-le et réessayez l'action.
  </Accordion>
  <Accordion title="Vérifier votre profil Chrome">
    Chrome stocke les passkeys par profil. Si vous êtes connecté à un profil Chrome différent de celui utilisé lors de la création du compte, votre passkey n'apparaîtra pas. Basculez vers le bon profil et réessayez.
  </Accordion>
  <Accordion title="Vérifier que vous êtes sur le bon appareil">
    Les passkeys sont liés à un appareil spécifique. Si le compte a été créé sur un ordinateur portable, le passkey ne sera pas disponible sur un téléphone, et inversement. Retournez à l'appareil d'origine.
  </Accordion>
  <Accordion title="Vérifier votre gestionnaire de mots de passe ou trousseau">
    Si vous utilisez un gestionnaire de mots de passe (comme 1Password ou Google Password Manager) ou le Trousseau iCloud d'Apple, assurez-vous qu'il est connecté et déverrouillé. Votre passkey peut y être stocké.
  </Accordion>
</AccordionGroup>

Si aucune de ces solutions ne résout le problème, une réinitialisation du passkey est nécessaire. Contactez l'équipe Byzantine à [info@byzantine.fi](mailto:info@byzantine.fi) pour commencer.

## Comment fonctionne la réinitialisation

L'équipe Byzantine peut configurer un nouveau passkey lors d'un court appel, généralement cinq minutes ou moins. Un appel est nécessaire pour des raisons de sécurité - votre passkey autorise les transactions sur votre compte, ce qui signifie que votre identité doit être confirmée au préalable.

<Warning>
Avant l'appel, assurez-vous d'être sur l'appareil et le profil de navigateur où vous souhaitez le nouveau passkey. Le passkey sera créé sur l'appareil et le profil utilisés lors de la réinitialisation.
</Warning>

<Steps>
  <Step title="Ouvrir la page de récupération">
    L'équipe Byzantine vous dirigera vers une page de récupération dans l'application pendant l'appel.
  </Step>
  <Step title="Entrer le premier code">
    Vous recevrez un code pendant l'appel. Collez-le dans la page de récupération.
  </Step>
  <Step title="Entrer le code reçu par email">
    Un second code arrivera par email. Collez-le également dans la page de récupération.
  </Step>
  <Step title="Configurer votre nouveau passkey">
    Votre navigateur vous invitera à créer un nouveau passkey. Confirmez avec votre empreinte digitale, votre reconnaissance faciale ou le code PIN de votre appareil, comme lors de la configuration initiale du compte.
  </Step>
</Steps>

Une fois les quatre étapes terminées, rechargez la page. Votre nouveau passkey est actif immédiatement.
